Group Administration

Sign in with your M365 admin account to add, remove, or export group members.

Requires User.Read.All, Group.ReadWrite.All, GroupMember.ReadWrite.All, and Directory.Read.All.

About
Group Administration Add, remove, or export members of Entra security groups and Microsoft 365 groups — live via Microsoft Graph. Distribution lists, mail-enabled security groups, and shared mailboxes generate a PowerShell script (coming soon).
How it works Pick an object type, choose add / remove / export, target the group, and (for add/remove) upload a CSV of email addresses or UPNs. Every change previews as a dry run before you go live, with a full audit log.
